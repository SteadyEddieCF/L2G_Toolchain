Local/offline L2G workbook loop

L2G Builder/Merger L2G-BM v3.8

A local/offline bridge for the full workbook loop: Workshop Handoff → generated or existing workbook → Workbook Merge return. v3.8 adds a persistent exception-resolution ledger, decision-plan import/export, an optional unresolved-decision execution gate, downloadable governance ZIP packs, and an embedded rolling release roadmap while preserving the v3.7 compatibility and three-stage diff model. Build mode uses the embedded L2G Builder Template v2 when no template is uploaded; Merge mode returns reviewed workbook results through l2g_workbook_merge_v1.json; External CSV mode remains a literal CSV-to-XLSX path.

Safety guardrail: Build mode adds workshop context only and does not auto-mark practices/objectives as Met, Not Met, Fully Implemented, Evidence Sufficient, Objective Validated, Workbook Validated, Assessment Ready, Final CMMC Status, or Final Score. Merge mode exports workbook-derived status because the source is the reviewed Gap Analysis Workbook.
Ready. Files stay in this browser process and are not transmitted.

Build Populated Workbook

Combines the Gap Workshop Tool Workbook Handoff into a copy of the official Gap Analysis Workbook so the reviewer can work directly in the workbook instead of comparing two large files side by side.

Input 1Blank/template Gap Analysis Workbook. Optional; embedded L2G Builder Template v2 is used when omitted.
Input 2l2g_workbook_handoff_v1.json or supported legacy handoff/state package.
OutputPopulated workbook copy plus a build report.

Compact mode uses v46-v54 population guidance first, writes only short notes to official workbook fields, populates L2G Advisor Review with concise finding rows, and keeps long source/provider/detail context on L2G Detail Register. Expanded mode adds separate L2G columns for deeper internal analysis. Helper sheets only keeps the original workbook tabs untouched.

Default behavior preserves existing reviewer-entered objective text. Select replacement only when the uploaded workbook should be refreshed from the v54 handoff.

L2G-BM preserves existing sheets and formulas, writes short official workbook notes only, populates L2G Advisor Review for concise review/report findings, uses L2G Detail Register for long source detail and traceability, and populates an existing JSON Import tab when present.

No workbook loaded. If you upload a Workbook Handoff without a template, the embedded L2G Builder Template v2 will be used.

Create Workbook from External CSV

Use this when a colleague provides a CSV from Glean, another AI tool, or another non-Workshop source and you simply need that data inside an XLSX copy of the blank CMMC workbook template. This path does not require the CMMC L2 Gap Workshop Tool and does not interpret or normalize the CSV values.

Input 1Blank/basic CMMC workbook template or the L2G Builder/Merger template.
Input 2External CSV from a colleague or approved internal tool.
OutputWorkbook copy with a visible External CSV Import sheet containing the CSV values as text.
Exact-copy behavior: this mode writes the CSV header row and data rows into a new worksheet as-is. It does not mark Met/Not Met, does not create scoring, does not summarize findings, and does not alter official Gap Analysis Details values.

1Upload blank/basic workbook template

The basic official workbook is allowed here. L2G-specific support tabs are not required for this external CSV path.

No template workbook loaded. If you upload an external CSV without a template, the embedded L2G Builder Template v2 will be used.

2Upload external CSV

CSV values are written to the workbook as text. This preserves AI-generated wording, statuses, confidence values, recommendations, source filenames, and notes without L2G interpretation.

No external CSV loaded.
