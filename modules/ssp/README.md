# CMMC L2 SSP Modern Editable

Local SSP authoring and Word Review round-trip module with Workshop handoff/return integration.

- Current supplied release candidate: **v1.8.8**
- Current release focus: **Consolidated change and decision traceability with deterministic source provenance and append-only explicit governance notes**
- Next bounded action: **v1.8.9 configurable inheritance/reconciliation policy profiles**

Single-System remains the default. Portfolio mode is opt-in and local-only. Every module retains exactly 110 explicit requirement records. v1.8.8 adds a deterministic consolidated register across changes, decisions, imports, Word Review, parent-change impacts, conflicts and exceptions, formal review, approvals, named baselines, and restoration. Authorized local roles may append scoped change or decision notes; explicit entries are not edited or deleted. The derived register does not rewrite its authoritative source records and does not create authenticated identity, digital signatures, assessment findings, certification decisions, readiness scores, or compliance conclusions.

Preserve existing SSP and Word-reviewed values at higher precedence than imported draft candidates. Preserve `l2g_ssp_handoff_v1` and `l2g_ssp_return_package_v1` at version 1.0. Portfolio/module exchange remains version 1.5 compatible, the formal review register remains version 1.6, immutable named baseline packages remain version 1.7, and consolidated change/decision register exports use version 1.8.
