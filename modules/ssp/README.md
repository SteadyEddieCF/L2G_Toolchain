# CMMC L2 SSP Modern Editable

Local SSP authoring and Word Review round-trip module with Workshop handoff/return integration.

- Current supplied release candidate: **v1.9.4**
- Current release focus: **Deterministic local portfolio change calendar and review-cycle planning with portfolio/module scope, JSON snapshot, CSV events, publish-only ICS, and delivery-package inclusion**
- Next bounded action: **v1.9.5 portfolio role coverage and responsibility matrix (suggested)**
