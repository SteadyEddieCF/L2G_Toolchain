# User and Administrator Guide — v1.8.8

Open **Actions → Manage SSP portfolio** and use **Consolidated change and decision register**.

1. Set the current actor and action role in Formal review and approval.
2. Choose register scope and entry type.
3. Record summary; decisions also require rationale.
4. Append the entry. Entries cannot be edited or deleted.
5. Filter the consolidated register by module, category, state, or search text.
6. Export the filtered view as JSON or CSV.

Administrators should periodically export the register with delivery packages. Source records remain authoritative when investigating an entry.
