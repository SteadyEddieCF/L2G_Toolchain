# CMMC L2 SSP Modern Editable v1.8.0

v1.8.0 is the first bounded multi-module foundation release. It keeps Single-System mode as the default and adds optional portfolio metadata, a stable module hierarchy, guided conversion, a complete pre-conversion rollback snapshot, versioned JSON schemas, and foundation exports.

Open `CMMC_L2_SSP_Modern_Editable_v1.8.0.html` locally in a modern browser. No server, install, account, telemetry, or network connection is required.

Important boundary: child modules are metadata shells in v1.8.0. Requirement applicability, inheritance, overrides, change propagation, consolidated CRM, conflicts, approvals, and baselines remain later releases.
