# User and Administrator Guide — v1.8.5

1. Keep Single-System mode unless a modular portfolio is required.
2. In portfolio mode, select **Export portfolio exchange JSON** for a complete portable package or **Export selected module exchange JSON** for one module.
3. Use **Import portfolio/module exchange JSON** and review safe, additive, unchanged, conflict, and invalid counts before applying.
4. Select a module and choose **Export selected module Word review**. Review in Word using comments and Track Changes.
5. Import the reviewed DOCX through **Import reviewed module Word**. Resolve its module queue; no feedback is applied automatically.
6. Export a backup before applying exchanges or review changes. Treat all outputs as authoring records, not assessment or certification conclusions.
