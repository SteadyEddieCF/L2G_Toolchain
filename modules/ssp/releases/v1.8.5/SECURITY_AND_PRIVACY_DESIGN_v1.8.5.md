# Security and Privacy Design — v1.8.5

All processing remains local and offline. Exchange fingerprints detect payload modification. Imports fail closed on contract, identity, catalog, hierarchy, and fingerprint errors. DOCX inspection rejects macros, ActiveX, embedded objects, unsafe paths, malformed XML, and oversized packages. Existing governed values are never silently overwritten. No telemetry, external upload, assessment result, or certification claim is created.
