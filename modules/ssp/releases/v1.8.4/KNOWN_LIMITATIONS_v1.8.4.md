# Known Limitations — v1.8.4

CRM is a derived authoring/reconciliation aid, not a formal approval, immutable baseline, assessment result, certification decision, or enterprise GRC system. The on-screen table renders the first 300 filtered rows for responsiveness, while CSV exports include all matching rows. Imported conflicts require human resolution outside the safe-only apply path. Module-specific Word Review and JSON package exchange remain planned for v1.8.5.
