# Release Notes — v1.8.4

## Added
- Consolidated 440-row CRM generation for the four-module McFirecoal fixture and 110-row selected-module CRM export.
- Authoritative implementation-source resolution and inherited/shared/supplement/override classifications.
- Separate counting dispositions for authoritative implementation, inherited references, shared obligations, supplements, and evidence.
- Filters for module, family, owner, status, classification, evidence, conflict, and approval metadata.
- CSV reconciliation for modular CRM v1 and the legacy single-system CRM format. Only empty/default, non-conflicting fields can be applied automatically.
- Persistent reconciliation summaries and source/legacy identifier provenance.

## Preserved
All 110 authoritative requirements, Single-System default, local/offline operation, Workshop handoff/return 1.0, Word Review v1, deterministic inheritance, parent-change review, and conflict escalation.
