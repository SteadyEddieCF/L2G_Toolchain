# User and Administrator Guide — v1.8.4

1. Open the HTML locally; continue in Single-System unless modular authoring is required.
2. In Modular Portfolio mode, open **Portfolio CRM Workbench**.
3. Filter the derived CRM or export the complete/selected-module CSV.
4. To reconcile a CRM, select the CSV and review safe, unchanged, conflicting, and invalid rows.
5. Export the reconciliation report. Apply only the non-conflicting rows after confirming the automatic backup download.
6. Resolve conflicts through governed author review; do not treat CRM output as an assessment conclusion.
