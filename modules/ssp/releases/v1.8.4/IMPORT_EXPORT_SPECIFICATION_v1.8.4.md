# Import and Export Specification — v1.8.4

The runtime continues to export/import full JSON backups and existing package contracts. CRM adds UTF-8 CSV exports for the full portfolio and selected module. CRM reconciliation accepts the v1.8.4 modular CRM columns or the supported legacy single-system CRM columns. Identity, catalog, derived classification, authoritative source, and existing governed values are protected. Only non-empty imported values targeting empty/default editable fields become safe updates. A backup is downloaded before UI application.
