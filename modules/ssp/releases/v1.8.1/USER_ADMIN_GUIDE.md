# User and Administrator Guide — v1.8.1

Use Single-System mode normally when one SSP is sufficient. In Portfolio mode, select a module in the hierarchy, then review its **Module applicability and responsibility** section.

Use search, family, and applicability filters to narrow the 110 records. **Unresolved queue** shows Pending decision records. Individual review is required for Not applicable because rationale is mandatory, and for inherited declarations because a source module is mandatory. Bulk review deliberately offers only values that do not require record-specific support.

Approval status is tracking metadata in v1.8.1, not an approval workflow. Export before destructive module deletion; deleting a leaf module also deletes its 110 decision records. Handle all exports as potentially sensitive security information.
