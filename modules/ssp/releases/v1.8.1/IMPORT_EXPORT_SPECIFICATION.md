# Import and Export Specification — v1.8.1

Accepted complete imports include existing SSP JSON through v1.8.0 and `cmmc_l2_ssp_portfolio_foundation_v1` packages containing an anchor workspace and foundation schema 1.0.0 or 1.1.0. Existing CRM CSV, Word Review DOCX, and Workshop handoff contracts remain unchanged.

Complete portfolio exports use package version 1.1 and include every module requirement decision plus counts and classification warnings. Selected-module exports include that module and its 110 decision records; the anchor export may also include the authoritative anchor workspace.

Module-specific import, modular CRM, consolidated CRM, and module-specific Word Review are deferred to their scheduled releases. Evidence fields contain references, not evidence binaries.
