## Start here

1. Open `CMMC_L2_SSP_Modern_Editable_v1.9.6.html` in a modern Chromium-, Firefox-, or WebKit-based browser.
2. Use **Author** for editing, **Review Document** for document-oriented reading, or **Focused Control** for one-control-at-a-time work.
3. Review the persistent document-state summary. “Saved in this browser” means browser-local storage only.
4. Use **Preflight** before delivery.
5. Use **Deliver** to choose an outcome and route to an existing export. A generated export or initiated download does not prove file retention.
6. Use **Advanced actions** for import, revision, recovery, optional Advanced portfolio mode, appearance, and destructive reset.

Single-System remains the default. Portfolio mode remains Advanced and optional.
