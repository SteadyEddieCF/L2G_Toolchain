# User and Administrator Guide — v1.8.3

Open **Parent-change impact review** to see stale records grouped by source module. Review the child snapshot and current source fields. **Refresh** explicitly accepts source values and clears a local override while preserving supplements. **Preserve** keeps the child snapshot, **Defer** leaves the decision queued, and **Escalate** creates an open conflict record. Enter a rationale for preserve, defer, or escalate. Export before broad adjudication.
