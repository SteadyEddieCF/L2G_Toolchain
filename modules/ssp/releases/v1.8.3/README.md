# CMMC L2 SSP Modern Editable v1.8.3

v1.8.3 adds grouped parent-change impact review, field-level before/after comparison, explicit refresh/preserve/defer decisions, and conflict escalation for local-override collisions. No source value propagates without an explicit author refresh.

Single-System remains the default. Portfolio mode remains optional and local-only. Workshop, SSP return, Word Review, and read-only audit contracts are unchanged.
