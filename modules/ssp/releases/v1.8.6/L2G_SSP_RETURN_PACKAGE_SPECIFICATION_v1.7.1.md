# L2G SSP Return Package Specification — v1.7.1

## Emitted contract
- `package_kind`: `l2g_ssp_return_package_v1`
- `package_version`: `1.0`
- intended consumer: CMMC L2 Gap Workshop Tool

## Content
- Engagement identity
- SSP document version, date, and owner
- Source-handoff and SSP-state fingerprints
- Current accepted authoring status for all 110 controls
- Review status for all 110 controls
- Accepted implementation, ownership, evidence, CSP/ESP, inheritance, scope, N/A, and POA&M fields
- Accepted source references retained from the Workshop handoff
- Per-control unresolved Word-review counts
- Aggregate review and conflict counts

## Exclusions and guardrails
- Reviewer-note text is excluded.
- Workshop preview is required.
- Existing Workshop values are not overwritten automatically.
- Status is labeled SSP authoring status, not assessment status.
- No final CMMC conclusions are emitted.
