# User and Administrator Guide — v1.8.6

1. Enable portfolio mode only when modular SSP management is needed.
2. In **Formal review and approval workflow**, enter the current actor name and select the action role.
3. If no roles exist, bootstrap that same actor as the first portfolio Administrator.
4. An Administrator assigns separate Author, Reviewer, and Approver roles at portfolio or module scope.
5. The Author submits the selected scope. A Reviewer records Ready for approval or Changes requested. A separate Approver records formal authoring approval.
6. If governed content changes, the approval remains recorded but displays as stale. Submit a new review round for fresh approval.
7. An Approver or Administrator may revoke an active approval with a required reason.
8. Export the review register JSON for offline governance evidence. Do not treat it as an assessment or certification result.
