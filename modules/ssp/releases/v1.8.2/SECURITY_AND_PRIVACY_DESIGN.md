# Security and Privacy Design — v1.8.2

Processing remains offline in the single HTML file. Imported values are normalized and length-limited. Module IDs, same-requirement pairs, hierarchy, source eligibility, inheritance cycles, source fingerprints, override rationale, and enumerations are validated. The FNV-1a fingerprint is a deterministic change detector, not a cryptographic signature or approval. Evidence remains reference-only; no telemetry, cloud backend, or automatic compliance conclusion is added.
