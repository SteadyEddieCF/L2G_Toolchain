# CMMC L2 SSP Modern Editable v1.8.2

v1.8.2 adds deterministic, same-requirement inheritance from eligible ancestor or active shared-service modules. It records canonical source fingerprints, marks parent changes stale without silent overwrite, separates local supplements from inherited base fields, and requires a rationale for governed local overrides.

Single-System remains the default. Portfolio mode remains optional and local-only. Workshop, SSP return, Word Review, and read-only audit contracts are unchanged.
