# User and Administrator Guide — v1.8.2

Select a child module and review a requirement. Choose inherited applicability, then select an eligible ancestor or shared-service source. Saving copies the fixed inherited field set and records a fingerprint. Use **inherited with local supplement** for residual module narrative, owners, or evidence. Use **governed local override** only with a documented rationale. When a parent changes, filter **Stale parent** and explicitly refresh or preserve the override. Export before broad changes. Single-System users need take no action.
