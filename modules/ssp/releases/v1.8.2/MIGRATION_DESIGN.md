# Migration Design — v1.8.2

v1.8.1 portfolio 1.1 declarations migrate to schema 1.2.0 without inventing fingerprints or copying source content. Existing inherited declarations begin `source-pending` until an author opens and saves the record against an eligible, ready same-requirement source. v1.8.1 autosave, images, Word Review queues, and rollback snapshots remain discoverable. Single-System data remains Single-System.
