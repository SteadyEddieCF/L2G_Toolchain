# L2G Toolchain

Local/offline CMMC advisory toolchain coordinated through independently versioned modules and stable package contracts.

## Modules

- L2G Control Center
- DocConverter-L2G
- L2G Scoper
- CMMC L2 Gap Workshop Tool
- L2G Builder/Merger
- CMMC L2 SSP Modern Editable

The canonical runtime remains portable, local HTML unless a later approved architecture decision changes it. The monorepo centralizes contract governance, fixtures, suite snapshots, automated validation, and portable-suite packaging while preserving module ownership and independent release cadence.

See `docs/architecture/L2G_Toolchain_Monorepo_Operating_Model_v1.md` and `CONTRIBUTING.md` before starting a module release.
